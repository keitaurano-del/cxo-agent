# スクリーニング採点基準

LLM に論文 1 本ずつ渡し、以下 4 軸を 1〜5 点で採点させる。
出力は JSON のみ (後段でパース)。

## 軸

| 軸 | 1 点 | 5 点 |
|---|---|---|
| 新規性 | SNS で繰り返し語られた既出ネタ | 国内ではほぼ未紹介 |
| 意外性 | 直感通りの結果 | 直感に反する驚きがある |
| 実用性 | 個人の行動に結びつかない | 明日から試せる具体策が出る |
| 信頼性 | 査読なし・サンプル小・再現性疑義 | 査読済 / 大規模 / メタ分析 |

## 出力フォーマット

```json
{
  "novelty": 1-5,
  "surprise": 1-5,
  "utility": 1-5,
  "credibility": 1-5,
  "total": <sum>,
  "one_liner": "<日本語 40 字以内のフック候補>",
  "skip_reason": "<除外時のみ理由を 1 行>"
}
```

## 除外条件 (即 skip)

- アブストラクトが取得できない
- 働き方・生産性・職場・睡眠×仕事・集中・休息のいずれにも関係しない
- 政治的・差別的に扱いが難しいテーマ
- 商業プロダクトの宣伝色が強い
